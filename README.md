# Superstore Sales Profitability Analysis
This project analyzes Superstore sales data to understand sales trends,proftability, and shipping performance.
## Dataset
Superstore Sales Profitability Dataset (Excel).
## Objectives
Analyse total sales and profit.
Identify profit by region.
Evaluate shipping performance.
Calculate customer lifetime value (CLV).
